# 한시트 SEO · 전환 전략 (요약)

> 이 사이트의 목적은 단 하나: **"포장영상" 관련 검색으로 유입 → zeropacking에 문의(리드)를 만드는 것.**
> 사이트가 돈을 버는 게 아니라 **문의를 넘겨주는 깔때기**다.

## 페이지별 키워드 주인 (겹치지 않게 나눈다)

| 페이지 | 노려야 할 검색어 (주인) |
|---|---|
| 홈 `/` | **포장영상, 포장영상 촬영, 포장영상 자동 촬영, 포장영상 서비스** (대표어) |
| `/guide` | 포장영상 도입, 포장영상 설치, 포장영상 카메라 위치 |
| `/vs-cctv` | 포장영상 vs cctv, 포장영상 cctv 차이 |
| `/compare` | 포장영상 업체 비교, 포장영상 추천 |
| `/pricing` | 포장영상 요금, 포장영상 가격 |
| `/cases` | 포장영상 효과, 포장영상 도입 사례 |
| `/claim` | 택배 클레임 해결, 빈박스 분쟁 |
| 블로그 각 글 | 쿠팡 클레임, 택배 파손 보상 등 개별 롱테일 |

**규칙: "포장영상"·"포장영상 촬영" 단독은 홈만 노린다. 나머지는 뒤에 단어를 붙여 차별화한다.**
새 페이지/글을 쓸 때 반드시 "이 글의 검색어 주인이 누구인지"를 먼저 정한다.

## 다국어
- 기본(한국어) 루트, 그 외 `/<locale>`: `/en` `/ja` `/zh-tw` `/vi`
- 각 언어 문의 창구: 한국 `zeropacking.com/#inquiry` · 일본 `/ja#inquiry` · 대만 `/zh-tw#inquiry` · 베트남 `/vi#inquiry`
- 블로그는 한국 시장 전용이라 번역하지 않는다. 해외 블로그는 현지 시장용 새 글로 쓴다.

## 지금부터 할 일 (코드가 아니라 이것들이 순위·매출을 만든다)
1. **트래픽** — 2페이지 → 1페이지는 **시간 + 백링크**가 핵심. 물류·이커머스 커뮤니티/블로그에서 홈으로 링크 확보. 수요 검증은 소액 광고로 1~2주 테스트.
2. **신뢰** — 실제 포장영상 검색 화면 캡처 1장, 실제 고객 후기 1~2개(가능하면 로고). 이게 언어 5개보다 전환에 크다. (현재 이미지는 예시/일러스트)
3. **측정** — 매달 "문의 몇 건 들어왔나"만 본다. 그게 유일하게 중요한 숫자.
4. **순서** — 한국에서 "문의→결제"가 도는 걸 먼저 확인하고, 그 공식을 해외에 복제한다.

## 기술적으로 이미 완료된 것 (더 손대지 않아도 됨)
- 페이지별 메타/구조화 데이터(JSON-LD), hreflang, 사이트맵/robots, 정적 생성(빠름)
- 전 페이지 상단 고정 문의 바 + 홈 히어로 "도입 문의하기" 주 버튼
- 발행일 고정(배포마다 날짜 점프 제거), 홈 타이틀 키워드 앞배치, WebPage 스키마

## 측정 도구
- Google Search Console: `sitemap.xml` 제출 후 "검색어/페이지"에서 실제 순위·노출 확인.
